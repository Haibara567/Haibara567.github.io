# Springboot 的优势

  • 通过简化配置可以快速搭建 Spring 应用程序。
  • 内嵌 servlet 容器，能够直接使用 java 的 main 方法启动，因此也不需要部署 war 包文件。
  • 提供 starter 简化 Maven 配置。

# 有监督学习和无监督学习的区别

有监督：通过已有的训练样本（即已知数据及其对应的输出）去训练得到一个最优模型

无监督：由输入数据中学到或建立一个模型，并依此模式推测新的结果。输入数据没有被标记，也没有确定的结果。
